# 内容と実験

報告時点ではいくつか状況が混在しているのでいくつかに切り分けた。

1.  陸上で(場所)
2.  ボートに乗った(乗る物)
3.  クリーパーが(爆発物)
4.  爆発すると、乗っているボートを破壊出来ない？

| 陸上 | 竹       | クリーパー | 破壊        |
| 水上 | 木製     | クリーパー | 破壊        |
| 水上 | 木製     | TNT        | 破壊        |

| 水上 | 竹       | クリーパー | 破壊        |
